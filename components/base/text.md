# text

文本。

**属性及支持度**

属性|类型|默认值|说明|微信|百度|快应用|字节跳动 
---|---|---|---|---|---|---| 
selectable | Boolean | false | 文本是否可选 | ✔️ | X | Ｘ | ✔️ 
space | String | false | 显示连续空格 | ✔️ | ✔️ | Ｘ | ✔️ 
decode | Boolean | false | 是否解码 | ✔️ | X | Ｘ | ✔️ 


space有效值：

值|说明
---|---|
ensp | 中文字符空格一半大小
emsp | 中文字符空格大小
nbsp | 根据字体设置的空格大小
